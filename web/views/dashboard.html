📖 Guide rapide — Produits

🏷️ 1. Création & Essentiels

Le Nom et le Prix sont les seuls champs obligatoires. Utilisez la Description pour préciser les détails techniques ou les saveurs.

📸 2. Galerie Médias

Glissez plusieurs photos ou vidéos. La première de la liste sera la couverture (miniature) visible sur Telegram.

🎁 3. Offres & Marketing

Packs : créez des offres "Achetez-en 2, recevez le 3ème gratuit". Dégressif : configurez des prix qui baissent selon la quantité.

🏪 4. Liaison Fournisseur

Liez ce produit à un Fournisseur pour qu'il reçoive les commandes automatiquement et puisse valider la préparation (Bouton "Prêt").

👁️ 5. Visibilité & Ordre

Glissez les cartes (≡) pour trier. Utilisez l'icône "œil" pour masquer un produit momentanément (ex: rupture de stock).

💡 Conseil : Allez dans Paramètres → Textes pour personnaliser les messages automatiques du bot.

🏪 Fournisseurs

📘 Guide Fournisseur : Comment ça marche ?

1️⃣ Inscription & Liaison

Ajoutez le fournisseur avec son Prénom/Nom et son ID Telegram (utilisez @userinfobot pour l'obtenir). Le compte doit être Actif ✅.

2️⃣ Gestion du Catalogue

Liez des produits à ce fournisseur dans l'onglet Produits. Il recevra alors une alerte automatique pour chaque commande client.

3️⃣ Workflow "Prêt"

Le fournisseur prépare le colis et clique sur "Prêt ✅" sur son bot. L'Admin est alerté pour envoyer un livreur récupérer la marchandise.

4️⃣ Mode Marketplace

Si le Marketplace est actif, le fournisseur peut poster ses propres produits en gros. Vous pouvez alors les copier dans votre vitrine client.

💡 Rappel Logistique : Dans le formulaire, choisissez si l'Admin gère la livraison finale ou si le Fournisseur s'en occupe directement.

➕ Nouveau Fournisseur

Définit qui livre quand un client achète ce produit dans le catalogue.

📜 Manuel & Guide Complet

🏪

Marketplace (B2B)

Permet de travailler avec des fournisseurs externes qui proposent leurs stocks.

Utilisation : Approuvez un fournisseur dans l'onglet "Fournisseurs". Son stock devient disponible dans le Marketplace interne. Utilisez le bouton ✨ Publier au Catalogue Client pour copier un produit vers votre boutique de détail.

Désactiver : Allez dans Paramètres > Sécurité et décochez "Marketplace".

🛍️

Catalogue & Retail

Gestion de vos produits de vente directe aux clients.

Utilisation : Gérez prix, stocks, catégories et photos. Les stocks à 0 masquent automatiquement le produit.

Modifier : Cliquez sur une carte produit pour éditer ses détails. Utilisez l'icône "œil" pour masquer/afficher sans supprimer.

🛵

Logistique & Suivi

Gestion des livreurs et du parcours de commande.

Utilisation : Les livreurs s'inscrivent via le bot. Une fois validés, ils reçoivent les alertes de commandes. Le bot gère le statut : Accepté > Préparé > Arrivé > Livré.

Secteurs : Les livreurs choisissent leur ville pour filtrer les commandes à proximité.

💰

Fidélité & Wallet

Système automatique de points et de cashback.

Utilisation : Chaque achat génère des points (ex: 1€ = 1pt). Le client convertit ses points en solde portefeuille utilisable pour payer ses commandes.

Configurer : Réglez les ratios de conversion dans Paramètres > Fidélité. Mettez le ratio à 0 pour désactiver.

📢

Diffusion (Broadcast)

Envoi massif de notifications sur Telegram et WhatsApp.

Utilisation : Onglet "Diffusion". Rédigez, joignez un média et lancez. Le système gère les pauses (Rate Limit) pour éviter les bannissements.

Stats : Suivez le nombre de messages délivrés et les utilisateurs ayant bloqué le bot.

🛡️

Sécurité & Blocage

Protection de votre plateforme contre les abus.

Utilisation : Onglet "Utilisateurs". Vous pouvez bloquer un client d'un clic. Ajoutez ou retirez des administrateurs (par ID Telegram) dans Paramètres > Sécurité.

Multi-Bot : Activez ou coupez indépendamment l'accès via Telegram ou WhatsApp.

🗨️ Système de Support Intégré

Lorsqu'un client demande de l'aide via le bouton "❓ Aide" du bot, vous recevez une alerte sur Telegram avec un bouton 💬 Répondre. En cliquant dessus, vous entrez en "Chat Direct" avec lui : chaque message que vous envoyez au bot sera transmis au client tant que vous ne coupez pas la session (ou 30 min après la dernière activité).

Bonus Paliers de Commandes

Offrez un bonus automatique dès qu'un client atteint un certain nombre de commandes.

💡 Comment ça marche ?
  • Fidélité : Le client cumule des points sur chaque commande. Il pourra les échanger contre l'argent dans son portefeuille (cagnotte).
  • Paliers : Dès que le client valide une commande dont le numéro est coché (ex: 5e ou 10e), le bot lui verse automatiquement le bonus.

Modes de Paiement

Configurez les options de paiement proposées lors de la confirmation de commande.

💡 Guide

L'ID doit être unique (ex: CASH, CARD, PAYPAL). L'icône et le libellé seront affichés sur les boutons du bot.

🛡️ Sécurité & Accès

Contrôle de l'approbation des nouveaux clients.

Si désactivé, tout nouveau client devra être validé manuellement par un admin.
Pour WhatsApp, désactivez cette option si vous voulez éviter l'envoi de messages non-sollicités (anti-ban).

🛒 Panier Abandonné

Envoyé automatiquement après 30 minutes d'inactivité.

📢 Abonnement Forcé

Les clients doivent rejoindre votre canal Telegram avant de commander.

L'ID doit commencer par -100. Le bot doit être admin du canal.

🚀 Livraison Prioritaire

Permet aux clients de payer un supplément pour passer en priorité.

ZONE DE DANGER (IRRÉVERSIBLE)

Attention : Ce bouton va supprimer DÉFINITIVEMENT tous les utilisateurs (clients et livreurs), toutes les commandes, tous les produits, toutes les statistiques et l'historique des diffusions. Vous allez TOUT PERDRE. À utiliser uniquement pour remettre le bot complètement à zéro.
